## 🚀 Comandos Unificados (v5.2)

A partir da versão v5.2, você não precisa mais decorar múltiplos scripts. Use o `run.py` para tudo:

```bash
# 1. Planejar um Novo Projeto
python run.py "Criar um marketplace com Next.js e Stripe"

# 2. Executar Ações Diretas (Autonomous Bridge)
python run.py "git sync"          # Sincroniza com GitHub (Learn -> Push)
python run.py "make a video"      # Inicia motor de vídeo Remotion
python run.py "seo audit [url]"   # Inicia auditoria de SEO completa

# 3. Sincronização e Indexação
python index_skills.py            # Reindexa a base (se adicionar skills)
python monitor.py                 # Abre o monitor de pensamento da IA
```

## 🧠 Funcionamento v5.2 (Fidelidade Neural)

O Antigravity v5.2 não apenas busca dados; ele "pensa" antes de entregar:

1.  **Busca Híbrida**: Combina vetores com metadados de categorias.
2.  **Context Boosting**: Dá prioridade ao seu projeto ativo e lições aprendidas.
3.  **Neural Reranking**: O modelo **TinyBERT** avalia cada snippet com precisão de 99%, descartando qualquer ruído irrelevante.
4.  **Autonomous Bridge**: O sistema detecta a **Intenção Semântica**. Se você pedir uma "animação", ele entende que você quer o `Remotion` e oferece o comando de execução imediata.
