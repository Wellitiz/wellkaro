### 2. ⚡ PEP v3.0 (Battle-Tested Performance Protocol)
O benchmark de performance forjado na batalha do projeto **Verdeon** (97 Mobile / 95 Desktop):
- **Regra Zero:** Providers NUNCA bloqueiam SSR com div vazia (previne NO_LCP fatal).
- **LCP-Safe Animations:** Apenas `transform` acima da dobra — conteúdo sempre visível.
- **Lazy 3D/Globe:** Three.js carrega apenas com IntersectionObserver, eliminando 650KB do bundle.
- **CSS Nativo:** ScrollReveal com CSS transitions, sem framer-motion no caminho crítico.
- **Build Checklist:** Validação obrigatória do HTML antes de todo deploy.
