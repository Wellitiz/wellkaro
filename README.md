---

## 🏛️ Três Pilares da Consciência Plena (v6.1)

### 1. ⚔️ Multi-Agent Team Debate
Diga adeus às soluções genéricas. No Antigravity, cada prompt atravessa uma mesa redonda técnica entre especialistas de elite:
- **CyberGuard** (Auditoria de Segurança)
- **NitroStream** (Otimização de Performance)
- **NexusPrime** (Arquitetura de Sistemas)
*O resultado? Um consenso técnico refinado que elimina erros de lógica e falhas de segurança antes mesmo do código ser escrito.*

### 2. 📂 Universal Skill Generator
Expansão infinita. Com a Ponte de Curadoria, você pode absorver qualquer tecnologia do GitHub para dentro do ecossistema. O sistema analisa vantagens e desvantagens e, sob sua confirmação, gera as habilidades necessárias para dominá-la.

### 3. ✍️ Autonomous Session Historian
Auto-documentação absoluta. Integramos um historiador que analisa mudanças de código e tarefas concluídas para redigir seus relatórios de sessão automaticamente. Seus arquivos `SESSAO.md` e `walkthrough.md` são mantidos sempre atualizados pelo sistema.

## ⚡ Início Rápido

### Instalação em 60 segundos:
1.  Clone o repositório: `git clone https://github.com/Wellitiz/Antigravity-Welltiz.git`
2.  Instale as dependências: `pip install chromadb sentence-transformers rich`
3.  Inicie o monitoramento: `python monitor.py` (em um terminal separado)
4.  Comande: `python run.py "sua tarefa aqui"`
