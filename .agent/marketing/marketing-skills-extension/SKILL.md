# Marketing Skills Extension

> **Extensões de CRO, Copywriting e Marketing da framework [coreyhaines31/marketingskills](https://github.com/coreyhaines31/marketingskills)**

## 🎯 O que adiciona

Este layer complementa as skills existentes de marketing com:

| Categoria | Skills | Uso |
|-----------|--------|-----|
| **CRO** | page-cro, signup-cro, form-cro | Otimizar conversões |
| **Copywriting** | copywriting, copy-editing | Escrever copy |
| **Email** | cold-email, email-sequence | Outbound marketing |
| **Analytics** | analytics-tracking, ab-test | Medição |

## 🔄 Comandos CRO

### Page CRO
```
Use quando: Otimizar landing pages, homepage, pricing page
Trigger: "optimize this page", "increase conversions"
```

### Signup Flow CRO
```
Use quando: Melhorar registro/cadastro
Trigger: "improve signup flow", "reduce drop-off"
```

### Form CRO
```
Use quando: Otimizar formulários de lead
Trigger: "optimize form", "form conversion"
```

## ✍️ Framework Copywriting

### Estrutura AIDA

```
A - Attention (Headline)
I - Interest (Problema)
D - Desire (Benefícios)
A - Action (CTA)
```

### Estrutura PAS

```
P - Problem (Identificar)
A - Agitation (Agitar)
S - Solution (Solução)
```

## 📧 Email Sequence Template

### Welcome Sequence (5 emails)
```
Email 1: Bem-vindo + o que esperar
Email 2: Resolver problema #1
Email 3: Social proof
Email 4: FAQ/preço
Email 5: Urgência + CTA
```

### Cold Email Template
```
Subject: [Gancho] + [specific detail]

Olá [Name],

Vi que [insight sobre empresa].
[Nós ajudamos empresas como X a resolver Y].

Quer 15 min para explorar?

[CTA claro]
```

## 📊 A/B Testing Framework

### Testar um de cada vez
- ❌ Múltiplas mudanças simultâneas
- ✅ Uma mudança por test

### Duração mínima
- B2B: 2-4 semanas
- B2C: 1-2 semanas
- Statistical significance: 95%+

### O que testar primeiro
1. Headlines
2. CTAs
3. Preços/posição
4. Imagens
