# Skill: Playwright (Autonomous Automation & Vision)

> **Automação de Browser de Nível Industrial para Testes e Auditoria.**

## 🎯 O que é
`playwright` é o framework líder para testes e automação web em Chromium, Firefox e WebKit. Ele permite interagir com qualquer página web de forma programática.

## 🔌 Integração no Core (Caminho v7.0)
Esta skill é a base para a futura **Auto-Cura** do Antigravity.
- **Uso Atual**: Auditorias de UI, testes de regressão visual e web scraping de alta precisão.
- **Uso v7.0**: O Antigravity usará o Playwright para "ver" o site renderizado e validar se a implementação física bate com o `DESIGN.md`.

## 📜 Regras de Consciência
- **Validação Cruzada**: Sempre que um site for "finalizado", use o Playwright para garantir que todos os links e interações funcionam.
