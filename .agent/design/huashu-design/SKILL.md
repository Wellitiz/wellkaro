# Skill: Huashu Design (Advanced Prototyping)

> **Filosofia de Design Nativa-HTML e Revisão 5D.**

## 🎯 O que é
`huashu-design` é uma skill de design de alta fidelidade que foca em prototipagem nativa, animações complexas e uma filosofia de revisão em 5 dimensões. Ela é agnóstica a agentes e permite exportação para diversos formatos, incluindo MP4.

## 🔌 Integração no Core
O Antigravity utiliza as **20 Filosofias de Design** do Huashu para fundamentar o debate interno entre os agentes. O **NexusPrime** usa esta skill para conduzir revisões críticas de UI em 5 dimensões antes da entrega final.

## 📜 Regras de Consciência
- **Debate Crítico**: Ativado em projetos que exigem "Ultra-Fidelidade" ou "Animações Premium".
