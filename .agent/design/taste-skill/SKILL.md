# Skill: Taste-Skill (Anti-Slop Guardian)

> **O Guardião do Bom Gosto. Combate o design genérico e entediante.**

## 🎯 O que é
`taste-skill` é uma camada de inteligência estética que impede a IA de gerar soluções de design preguiçosas, clichês ou "slop" (conteúdo gerado sem alma).

## 🔌 Integração no Core
Esta é a skill favorita do **NitroStream**. Ela atua como um filtro obrigatório em todas as gerações de UI do Antigravity. Se o código gerado for considerado "básico demais", esta skill força uma re-iteração com foco em elegância e originalidade.

## 📜 Regras de Consciência
- **Zero Genericidade**: Proibido usar layouts de "bootstrap padrão" ou cores saturadas primárias sem refinamento.
