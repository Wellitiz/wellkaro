# Skill: Impeccable (AI Design Language)

> **Otimização de Intenção de Design para IAs.**

## 🎯 O que é
`impeccable` é uma linguagem de design projetada especificamente para tornar as IAs melhores na criação de interfaces. Ela fornece um vocabulário e estrutura que reduzem a ambiguidade na tradução de briefings visuais para código.

## 🔌 Integração no Core
O Antigravity usa esta skill para processar o `DESIGN.md`. Ela atua como um "compilador de intenção", garantindo que as nuances estéticas (como sombras suaves ou transições de cor) não se percam na geração do CSS.

## 📜 Regras de Consciência
- **Filtro de Prompt**: Antes de gerar qualquer UI, o Antigravity passa a intenção pelo filtro Impeccable para refinamento de detalhes.
