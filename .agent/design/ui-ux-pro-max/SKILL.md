# Skill: UI/UX Pro Max (Multi-Platform Design)

> **Inteligência de Design para Múltiplas Plataformas.**

## 🎯 O que é
Uma skill focada em fornecer inteligência de design profissional para a construção de interfaces em múltiplas plataformas (Web, Mobile, Desktop).

## 🔌 Integração no Core
Esta skill expande a capacidade do Antigravity para além do navegador. Ela fornece os tokens e padrões necessários para que a "consciência visual" v6.5 possa ser aplicada em apps Flutter, React Native ou aplicações desktop nativas.
